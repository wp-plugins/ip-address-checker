IP Address Widget displays user information (IP address, browser, operating system, country) in your blog.

== Description ==

The Widget displays following user information:

*		IP Address
*   Browser
*   Operating system
*   Country

It is multilingual.

== Installation ==

To use the widget you only have to do this:

1. Upload `ip-checker.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin through the 'Widgets' menu in WordPress

== Frequently Asked Questions ==

= Which languages are supported? =

The banner will be displayed in the user's language. Following languages are supported:
* English
* German
English is default language.
